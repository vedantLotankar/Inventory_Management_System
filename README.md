# 📦 Inventory Management System  

Welcome to the **Inventory Management System** repository! 🏪 This project is designed to manage inventory efficiently by providing basic **CRUD (Create, Read, Update, Delete) operations** using **Java Swing** for the front end and **MySQL** as the backend database.  

<br/>

## 🎯 Project Overview  

This system helps businesses keep track of their inventory by allowing users to add, modify, and remove items while storing data securely in a relational database. The application provides a user-friendly interface using **Java Swing** and ensures smooth data management with **MySQL**.  

- **Language Used:** Java ☕  
- **GUI Framework:** Swing 🎨  
- **Database:** MySQL 🗄️  
- **Operations:** CRUD (Create, Read, Update, Delete)  

<br/>

## 🛠️ Features  

✅ **Add New Inventory Items**  
✅ **View and Search Products**  
✅ **Update Stock Information**  
✅ **Delete Outdated Items**  
✅ **User-friendly Swing-based UI**  
✅ **Secure MySQL Database Integration**  

<br/>

## 🚀 How to Run  

1. Clone the repository:  
   ```bash
   git clone https://github.com/vedantLotankar/Inventory_Management_System.git
   ```  
2. Install dependencies and configure the MySQL database.  
3. Run the Java application:  
   ```bash
   javac Main.java
   java Main
   ```
